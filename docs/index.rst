ME-ICA: Multi-Echo Independent Components Analysis
==================================================

This pipeline is developed as open-source software by `Elizabeth DuPre`_ and `Prantik Kundu`_.

.. _Elizabeth DuPre: http://emdupre.me/
.. _Prantik Kundu: http://www.mountsinai.org/profiles/prantik-kundu

About
=====

``meica`` is a pipeline for the preprocessing and analysis of acquired multi-echo fMRI data.
